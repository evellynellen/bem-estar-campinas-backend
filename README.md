# Bem Estar Backend

Bem Estar Backend

## Tecnologias

- Node.js
- TypeScript
- PrismaORM
- SQLite

## Setup da aplicacao

### Instalar dependencias

- npm install

### Setup do prisma

- npm run postinstall

## Iniciar a aplicacao

### Local

- npm run start:dev

## Comandos Auxiliares

### Gerar codigo SQL para criar a estrutura do banco de dados

- npx prisma migrate dev --name create-tables

### Visualizar o banco de dados (local)

- npx prisma studio
